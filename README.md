# felixliou.github.io
---
## 介绍
---
这个仓库主要用于 github 服务器搭建自己的个人博客，仓库名即为自己博客网站的[域名](https://felixliou.github.io)。个人的网站框架采用 hexo 框架，主题采用了较为常用的 [NexT](https://theme-next.iissnan.com) 主题，该主题界面简单、框架明了，同时又提供了较为丰富的个性定制和第三方插件服务集成，比如评论系统，数据统计与分析，搜索服务等。
